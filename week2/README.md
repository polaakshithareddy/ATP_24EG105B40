# Week 2: Advanced JavaScript & Basic API Concepts

This directory contains advanced JavaScript assignments, modular mini-projects, and an introduction to backend development using Express.js. To make things cleaner, the date-based folders have been restructured into logic-based topics.

## Folder Structure

### 1. `javascript-advanced/`
Contains standalone scripts demonstrating object/array manipulation and real-world system logic.
- **Object Concepts**: `object-deep-copy.js`, `object-shallow-copy.js`
- **Spread/Rest Operators**: `spread-operator-array-copy.js`, `spread-operator-object-update.js`, `rest-operator-sum-function.js`
- **System Logic Simulators**: 
  - `bank-transaction-system.js`
  - `employee-payroll-system.js`
  - `exam-portal-simulator.js`
  - `library-management-system.js`
  - `movie-streaming-platform.js`
  - `online-course-enrollment.js`
  - `otp-simulator.js`
  - `shopping-cart-system.js`
  - `student-marks-calculator.js`
  - `student-performance-evaluator.js`
  - `temperature-converter.js`

### 2. `mini-projects/`
Contains multi-file logical projects.
- **`e-commerce-system/`**: Simulates an e-commerce platform using multiple modules (`cart.js`, `product.js`, `payment.js`, etc.).
- **`task-management-system/`**: Simulates task handling and validation logic (`app.js`, `task.js`, `validator.js`).

### 3. `express-api-basics/`
An introductory Express.js backend setup.
- Contains `server.js`, `package.json`, HTTP request testing (`req.http`), and initial API routing setups.

## Concepts Covered
- Deep vs. Shallow object copying
- ES6 Spread (`...`) and Rest operators
- Real-world logic simulations (e.g. banking, grading, shopping carts)
- Modular logic scaling (Mini-projects)
- Backend initialization with Node/Express
