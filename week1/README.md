# Week 1: JavaScript Fundamentals

This directory contains introductory JavaScript assignments and exercises covering basic language fundamentals, conditionals, loops, functions, and array operations.

## Files and Descriptions

- **`employee-array-operations.js`** (formerly `Employee.js`): Demonstrates array operations (like `splice`) to insert, remove, and modify objects inside an array of employees.
- **`sum-array-elements-function.js`** (formerly `FunctionArgument.js`): Contains a function to calculate and return the sum of all elements passed in an array.
- **`greatest-of-three-function.js`** (formerly `Greatest3.js`): A function that takes three numbers as arguments and returns the greatest among them.
- **`largest-of-two.js`** (formerly `Largest2.js`): A simple script to find the largest of two initialized numbers.
- **`largest-of-three.js`** (formerly `Largest3.js`): A script using `if-else if-else` conditions to find the greatest of three variables.
- **`search-array-element.js`** (formerly `SearchElement.js`): Implements a function to search for a specific element in an array and print its index if found.
- **`smallest-array-element.js`** (formerly `SmallestOfArray.js`): Finds and prints the smallest value in an array by iterating through its elements.
- **`sum-of-marks.js`** (formerly `SumOfMarks.js`): Iterates over an array of marks to calculate and print the total sum.

## Concepts Covered
- Variable declarations (`let`, `const`)
- Conditional statements (`if`, `else if`, `else`)
- Loops (`for` loops)
- Functions (declarations and parameters)
- Arrays and Array Methods (indexing, `length`, `splice`)
- Template literals
