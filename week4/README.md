# Week 4: HTML Foundations & Semantic Structure

This directory contains introductory HTML files that have been refactored to emphasize semantic web structure and modularity.

## Files
- **`pochi-cat-profile.html`**: A profile page for a cat demonstrating semantic layout using `<header>`, `<main>`, `<section>`, and `<footer>`. It includes basic text formatting, lists, an image, and external links.
- **`html-basics.html`**: A beginner webpage demonstrating paragraphs, emphasis (`<strong>`), ordered/unordered lists, and hyperlink best practices.

## Improvements Made (Review Addressing)
Based on the review ("modularity and structure need improvement"):
1. **Semantic HTML:** Replaced generic layout structures with HTML5 semantic tags (`<header>`, `<main>`, `<section>`, `<footer>`) to improve accessibility and structure.
2. **Proper Document Outline:** Added missing `<!DOCTYPE html>` declarations and updated `<title>` tags to be descriptive instead of "Document".
3. **Accessibility:** Added descriptive `alt` tags to images and `target="_blank" rel="noopener noreferrer"` to external links for better security and user experience.
4. **File Organization:** Renamed generic files (`example.html`, `groupele.html`) to descriptive, kebab-case file names and removed date-based nested folders for a flatter, more modular project structure.
